# Comparador de Precios Mérida - LatinBien vs Competencia

Sistema de comparación de precios para Mérida, Venezuela. Compara precios entre LatinBien, SoyTechno, Multimax, MegaTecno y otras tiendas.

## 📊 Datos Verificados

Este proyecto contiene **40 productos verificados** de LatinBien con comparaciones de precios de otras tiendas cuando están disponibles.

- **Televisores**: Aiwa 32", 43", Samsung 65" 4K

- **Laptops**: All in One HP, Lenovo
- **Computación**: Impresoras Epson, Proyectores Epson
- **Routers**: Mikrotik, Starlink
- **Electrodomésticos**: Aires AC, Neveras, Cocinas, Lavadoras
- **Energía**: EcoFlow Delta 2, River 2

## 🚀 Uso

1. Abre `index_final.html` en tu navegador
2. Usa el buscador para encontrar productos
3. Filtra por categoría
4. Ve los precios comparados entre tiendas
5. Los mejores precios se resaltan en verde

## 📁 Archivos Principales

- `index_final.html` - Visualización principal (100% verificada)

- `scraper_latinbien.py` - Scraper para extraer productos de LatinBien
